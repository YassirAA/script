# HACKATHON

## PREPARACIO
  ### AWSCLI
  El primer de tot es instalar i configurar el aws
  ``` bash
  sudo apt update && sudo apt upgrade -y
  sudo apt install curl unzip -y
  curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
  unzip awscliv2.zip
  sudo ./aws/install
  aws --version
  ```
